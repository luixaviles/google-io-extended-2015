AngularJS Full-Stack generator

Prerequisites

                
                    $ npm install -g yo
                    $ npm install -g grunt-cli
                    $ npm install -g bower
                    $ npm install -g generator-angular-fullstack
                    $ mkdir my-project && cd $_
                    $ yo angular-fullstack

AngularJS Full-Stack generator

You can use different generators to start coding

                
                    $ yo angular-fullstack:route my-route
                    $ yo angular-fullstack:controller my-controller
                    $ yo angular-fullstack:service my-service
                    $ yo angular-fullstack:directive my-directive
